Eliminar usuario

Para eliminar un usuario se debe hacer clic sobre el botón de Eliminar, correspondiente al usuario que se desea eliminar Luego de haber hecho clic sobre este botón, se abrirá la siguiente ventana, en la cuan se muestra el número de cédula del usuario a eliminar.

Eliminar usuario

Vas a eliminar el usuario con número de identificación 123, esto se hará de forma permanente en la based e datos.

Si se desea eliminar el usuario, se hace clic sobre el botón de eliminar, de lo contrario se hace clic en cancela o cerrar para no eliminar el usuario.
Nota: si se elimina un usuario, este se eliminara de forma permanente de la base de datos.
Al eliminar un proveedor se mostrara el siguiente mensaje de satisfacción.

Usuario eliminado con éxito.
