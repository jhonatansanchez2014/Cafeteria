Agregar usuario

Para agregar un usuario, sellena el formulario correspondiente.

Agregar usuario






Luego de haber ingresado todos los datos, se debe hacer clic sobre el botón de guardar, situado en la parte inferior del formulario. Si el proceso es exitoso se mostrara el siguiente mensaje de satisfacción, y los campos del formulario se limpian de forma automática.

Usuario registrado con éxito.

Si ocurre algún error al momento de ingresar un usuario, se mostrara el siguiente mensaje, y los campos del formulario continuaran como los dejas para así puedas revisar los datos nuevamente.

El nombre de usuario o documento que está intentando de ingresar ya existe.

Al ingresar un usuario de forma exitosa, este se mostrara automáticamente en un formulario los datos ingresados.
