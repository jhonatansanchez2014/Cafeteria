Agregar usuario

Para editar un usuario de hace clic sobre el botón de editar correspondiente al usuario al cual se desea realizar cambios. Al hacer clic sobre este botón se abrirá la siguiente ventana

Actualizar datos del proveedor

Luego de haber realizado los cambios en los datos del usuario, se hace clic en el botón de guardar cambios, si el registro es exitoso, se muestra el siguiente mensaje

Usuario actualizado con éxito.

Si ocurre algún error al momento de actualizar los datos del usuario, se mostrara el siguiente mensaje, y los campos del formulario continuaran como los dejas para así puedas revisar los datos nuevamente.

El nombre de usuario o documento que está intentando de ingresar ya existe.

Al actualizar un usuario de forma exitosa, este se mostrara automáticamente en un formulario los datos ingresados.
