Control de acceso

Para acceder a la aplicación debe ingresar el nombre de usuario y contraseña asignadas en el siguiente formulario, y luego hacer clic en el botón de iniciar sesión.

Si el usuario o contraseña que se ingresó se mostrara el siguiente mensaje.
El Usuario y/o Contraseña ingresado son incorrectos, vuelva a intentarlo.

Si el usuario que intenta ingresar está bloqueado se mostrara el siguiente mensaje.
Lo siento, este usuario está suspendido y no puede acceder a la plataforma.

Si el usuario y contraseña ingresado son correctos, se permitirá el acceso inmediato a la aplicación conforme a los permisos que tenga el usuario.
